## Quick Setup

Follow these three simple steps to get your documentation live:

<Steps>
  <Step title="Clone the Repository">
    Start by cloning our starter template:
    ```bash
    git clone https://github.com/your-org/docs-template.git
    cd docs-template
    ```
  </Step>
  
  <Step title="Install Dependencies">
    Install the Mintlify CLI globally:
    ```bash
    npm i -g mint
    ```
  </Step>
  
  <Step title="Run Locally">
    Start the development server:
    ```bash
    mint dev
    ```
    Your docs will be available at `http://localhost:3000`
  </Step>
</Steps>

## Project Structure

Understanding the project structure will help you navigate and customize your documentation:

```
docs/
├── api-reference/     # API documentation
├── essentials/        # Core documentation guides
├── images/           # Static images and assets
├── snippets/         # Reusable content snippets
├── docs.json         # Main configuration file
└── index.mdx         # Homepage
```

## Configuration Basics

Your `docs.json` file controls the main settings:

```json
{
  "name": "Your Docs",
  "logo": {
    "dark": "/logo/dark.svg",
    "light": "/logo/light.svg"
  },
  "favicon": "/favicon.svg",
  "colors": {
    "primary": "#0D9373",
    "light": "#07C983",
    "dark": "#0D9373"
  }
}
```
